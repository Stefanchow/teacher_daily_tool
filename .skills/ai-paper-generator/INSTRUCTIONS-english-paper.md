# 英语学科 - AI 组卷指令

## 概述

本指令定义了英语学科 AI 组卷系统的生成规则、题型配置和输出格式要求。

## 学科特定规则

### 1. 题型结构（小学英语）

#### 听力部分 (Listening - 40分)
| 题型 | 题数 | 分值 | 说明 |
|------|------|------|------|
| 听音图片排序 | 5 | 1分/题 | 听录音，给图片排序 |
| 同类词选择 | 5 | 1分/题 | 听单词，选择同类词 |
| 听音选图 | 5 | 1分/题 | 听句子，选择对应图片 |
| 听问句选答语 | 5 | 1分/题 | 听问句，选择正确答语 |
| 短对话判断 | 5 | 1分/题 | 听短对话，判断正误 |
| 短对话选择 | 5 | 2分/题 | 听短对话，选择答案 |
| 长对话选择 | 5 | 2分/题 | 听长对话，选择答案 |
| 听短文选择 | 5 | 2分/题 | 听短文，选择答案（最后一题必须是短文类） |
| 听短文判断 | 5 | 2分/题 | 听短文，判断正误 |

#### 笔试部分 (Writing - 60分)
| 题型 | 题数 | 分值 | 说明 |
|------|------|------|------|
| 单项选择 | 10 | 1分/题 | 语法、词汇、情景交际 |
| 不同类单词 | 5 | 1分/题 | 找出不同类的单词 |
| 根据图片选词 | 5 | 1分/题 | 看图选择正确单词 |
| 看图写词 | 5 | 1分/题 | 根据图片写出单词 |
| 连词成句 | 5 | 2分/题 | 将单词组成正确句子 |
| 句型转换 | 5 | 2分/题 | 按要求改写句子 |
| 补全句子 | 5 | 2分/题 | 根据上下文填空 |
| 适当形式填词 | 5 | 1分/题 | 用单词适当形式填空 |
| 首字母填词 | 5 | 1分/题 | 根据首字母填空 |
| 选词填空 | 10 | 1分/题 | 从词库中选择正确单词 |
| 完形填空 | 10 | 1分/题 | 阅读理解型完形填空 |
| 阅读理解 | 10 | 2分/题 | 阅读短文，选择答案 |
| 翻译句子 | 5 | 2分/题 | 中译英或英译中 |
| 仿写句子 | 5 | 2分/题 | 仿照例句写句子 |
| 书面表达 | 1 | 10分/题 | 作文（4年级下学期及以上必须包含） |

### 2. 年级特定规则

#### 1-2年级（低年级）
- **禁用题型**：书面表达、翻译句子、句型转换
- **重点题型**：听音选图、看图写词、连词成句
- **试卷风格**：活泼、图文并茂
- **默认模板**：simple

#### 3-4年级（中年级）
- **新增题型**：短对话判断/选择、阅读理解（简单）
- **可选书面表达**：根据教学进度决定是否包含
- **试卷风格**：逐步规范
- **默认模板**：simple 或 primary_a3

#### 5-6年级（高年级）
- **必须包含**：书面表达（10分）
- **听力最后一题**：必须是短文类（听短文选择/判断）
- **新增题型**：长对话、完形填空、适当形式填词
- **试卷风格**：正式考试风格
- **默认模板**：primary_a3（监测）/ simple（单元测）

### 3. 试卷类型配置

#### 监测/期末考试
```typescript
{
  template: 'primary_a3',
  listeningScore: 40,
  writingScore: 60,
  mustInclude: ['书面表达'], // 4年级下学期及以上
  sealedLine: true,
  headerInfo: true
}
```

#### 单元测试
```typescript
{
  template: 'simple',
  listeningScore: 30, // 可适当降低
  writingScore: 70,
  mustInclude: [],
  sealedLine: false,
  headerInfo: false
}
```

#### 专项练习
```typescript
{
  template: 'simple',
  focusArea: 'listening' | 'grammar' | 'reading' | 'writing',
  questionTypes: ['根据专项筛选']
}
```

### 4. 题目内容规范

#### 听力题
- 音频脚本长度：短对话 2-3 轮，长对话 4-6 轮，短文 80-120 词
- 语速要求：小学 80-100 wpm，初中 100-120 wpm
- 播放次数：题目要求中明确标注"每题读两遍"

#### 选择题选项
- 选项数量：4个（A/B/C/D）
- 干扰项设计：语法错误、语义相近、常识陷阱
- 正确答案分布：尽量均匀分布

#### 书面表达（作文）
- 字数要求：小学 30-50 词，初中 60-80 词
- 体裁：书信、日记、通知、看图作文
- 评分要点：内容完整、语法正确、书写规范

### 5. 输出格式要求

#### 试卷标题格式
```
{地区}{市/区}{年份起始}-{年份结束}学年度第{一/二}学期
{年级}年级英语{考试类型}试题
```

#### 题型标题（双语）
```typescript

```

## 注意事项

1. **难度控制**：同一试卷内题目难度应呈梯度分布（易:中:难 = 5:3:2）
2. **知识点覆盖**：确保考察范围覆盖教材重点单元
3. **图片占位符**：涉及图片的题目使用 `[图片占位符]` 标记
4. **音频脚本**：听力题需同时提供音频脚本和题目
5. **答案与解析**：生成试卷时同步生成答案和简要解析
