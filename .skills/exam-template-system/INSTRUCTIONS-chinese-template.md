# 语文学科 - 试卷模板指令

## 概述

本指令定义了语文学科试卷的模板渲染规则。⚠️ **注意：语文板块正在进行教学法优化，当前版本为临时方案。**

## 状态说明

- **当前状态**：待优化
- **计划优化内容**：
  - 古诗文排版的特殊要求
  - 作文方格纸的设计规范
  - 阅读理解题的分栏布局
  - 文言文竖排支持（如有需要）

## 临时指令

在正式优化完成前，请使用通用模板规则渲染语文试卷。

### 基础模板建议
- **小学**：simple 模板（活泼风格）
- **初中/高中**：zhongkao 或 haidian 模板

### 特殊元素提醒
- 作文区域需要使用中文方格纸
- 古诗文可能需要特殊排版
