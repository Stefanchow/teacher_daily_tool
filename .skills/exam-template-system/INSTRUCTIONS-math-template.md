# 数学学科 - 试卷模板指令

## 概述

本指令定义了数学学科试卷的模板渲染规则。⚠️ **注意：数学板块正在进行教学法优化，当前版本为临时方案。**

## 状态说明

- **当前状态**：待优化
- **计划优化内容**：
  - 公式排版支持（LaTeX/MathJax）
  - 几何图形的占位符设计
  - 解答题的步骤分标注区域
  - 统计图表的展示方式

## 临时指令

在正式优化完成前，请使用通用模板规则渲染数学试卷。

### 基础模板建议
- **小学**：simple 模板（图文并茂）
- **初中/高中**：zhongkao 或 haidian 模板

### 特殊元素提醒
- 几何题预留图形绘制区域
- 解答题预留充足的作答空间
- 公式需要清晰可读
