# 语文学科 - 教案生成指令

## 概述

本指令定义了语文学科教案的生成规则。⚠️ **注意：语文板块正在进行教学法优化，当前版本为临时方案。**

## 状态说明

- **当前状态**：待优化
- **计划优化内容**：
  - 语文核心素养导向的教学设计
  - 阅读教学的不同课型（精读、略读、导读）
  - 写作教学的过程指导
  - 古诗文教学的文化渗透
  - 口语交际的情境创设

## 临时指令

在正式优化完成前，请使用通用教案模板生成语文教案。

### 基础教案结构
1. 教学目标（知识与能力、过程与方法、情感态度与价值观）
2. 教学重难点
3. 教学方法
4. 教学准备
5. 教学过程（导入-新授-巩固-小结-作业）
6. 板书设计
7. 教学反思

### 语文学科提醒
- 注重朗读指导
- 关注语言文字的品味
- 重视思维训练
- 渗透文化元素
