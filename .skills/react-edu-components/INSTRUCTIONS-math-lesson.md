# 数学学科 - 教案生成指令

## 概述

本指令定义了数学学科教案的生成规则。⚠️ **注意：数学板块正在进行教学法优化，当前版本为临时方案。**

## 状态说明

- **当前状态**：待优化
- **计划优化内容**：
  - 数学思维培养的教学设计
  - 问题解决的探究式学习
  - 概念教学的建构过程
  - 数学建模活动设计

## 临时指令

在正式优化完成前，请使用通用教案模板生成数学教案。

### 基础教案结构
1. 教学目标（知识与技能、过程与方法、情感态度与价值观）
2. 教学重难点
3. 教学方法
4. 教学准备（教具、学具）
5. 教学过程（情境导入-探究新知-巩固练习-课堂小结-布置作业）
6. 板书设计
7. 教学反思

### 数学学科提醒
- 注重数学概念的建构过程
- 关注数学思想方法的渗透
- 重视学生思维的可视化
- 设计有层次的练习题
